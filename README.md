# Sistema de Gestión de Mantenimiento Hospitalario

## Descripción
Aplicación dinámica para gestión de mantenimiento correctivo y preventivo 
de dispositivos médicos en hospitales, con base de datos en Supabase 
y aplicación móvil desarrollada en Android Studio.

## Tecnologías utilizadas
- Python
- Supabase (base de datos)
- Android Studio (aplicación móvil)
- API REST

## Roles y permisos
- Gerente General: acceso a todas las pantallas excepto programación de mantenimientos
- Ingeniero Biomédico: acceso completo a todas las plataformas
- Médico: acceso a estado de máquinas, mantenimientos y personal
- Paciente/Usuario: acceso a visualización de inventario de máquinas

## Funcionalidades
- Gestión de inventario de dispositivos médicos
- Programación de mantenimientos correctivos y preventivos
- Gestión de personal hospitalario
- Sincronización en tiempo real con base de datos en la nube
- Control de acceso por roles
